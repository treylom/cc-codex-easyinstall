# 윈도우 설치 가이드 (처음 쓰는 분도 그대로 따라오세요)

> 이 강의 환경은 **WSL(윈도우 안의 리눅스) + Claude Code + Codex + tmux**입니다.
> 아래 순서대로만 하면 됩니다. 명령어를 직접 칠 일은 거의 없어요 — **클릭과 [예]** 위주입니다.
> ⏱️ 전체 약 15~30분 (인터넷 속도에 따라) + 재부팅 1회.

---

## 0단계. 미리 알아둘 것 (딱 3가지)

- **재부팅 1번** 합니다. 재부팅 후 **자동으로 이어서** 설치되니 컴퓨터만 켜두면 됩니다.
- 중간에 **이름·비밀번호**를 한 번 만듭니다(우분투용). 까먹지 않게 적어두세요.
- 마지막에 **브라우저로 로그인** 한 번 합니다(Claude).

---

## 1단계. 설치파일 받기 (GitHub)

1. 강의에서 안내한 주소를 브라우저에서 엽니다:
   👉 `https://github.com/treylom/cc-codex-easyinstall`  *(최종 주소는 강의 슬라이드 기준)*
2. 페이지 오른쪽 위 **초록색 `< > Code` 버튼**을 클릭합니다.
3. 메뉴에서 맨 아래 **`Download ZIP`** 을 클릭합니다.  *(git 같은 건 몰라도 됩니다 — 그냥 ZIP 받기)*
4. 다운로드된 파일(예: `cc-codex-easyinstall-main.zip`)을 찾습니다 — 보통 **다운로드** 폴더.

> 📷 [스크린샷 자리: Code → Download ZIP]

---

## 2단계. 압축 풀기

1. 받은 ZIP 파일을 **마우스 오른쪽 클릭** → **`압축 풀기(Extract All)`** 를 누릅니다.
2. **`압축 풀기`** 버튼을 한 번 더 누르면, 같은 이름의 폴더가 생깁니다.
3. 그 폴더로 **들어갑니다**. 안에 **`설치.bat`** 파일이 보이면 성공.

> ⚠️ ZIP 안에서 바로 실행하지 말고 **꼭 압축을 푼 뒤** 실행하세요.

> 📷 [스크린샷 자리: 우클릭 → 압축 풀기 / 폴더 안 설치.bat]

---

## 3단계. 설치 실행 (`설치.bat` 더블클릭)

1. **`설치.bat`** 파일을 **더블클릭**합니다.
2. 파란 화면 **"Windows의 PC 보호 (Windows protected your PC)"** 가 뜨면:
   - **`추가 정보(More info)`** 클릭 → 아래에 나타나는 **`실행(Run anyway)`** 클릭.
   - *(이건 우리 설치파일이 아직 '서명' 등록 전이라 뜨는 정상 경고입니다.)*
3. **"이 앱이 디바이스를 변경하도록 허용하시겠습니까?"** (관리자 권한) 창 → **`예`** 클릭.

> 📷 [스크린샷 자리: SmartScreen 추가정보→실행 / UAC 예]

---

## 4단계. WSL 설치 + 재부팅 (자동)

1. 검은 창에서 윈도우 기능(WSL)과 우분투를 자동으로 받습니다. 몇 분 걸립니다.
2. **"지금 재부팅할까요? (Y/N)"** 가 나오면 **`Y`** 입력 후 Enter → 컴퓨터가 재부팅됩니다.
3. 재부팅 후 로그인하면 **자동으로 설치가 이어집니다**. (검은 창이 다시 떠요 — 닫지 마세요.)

> 💡 혹시 자동으로 안 이어지면, 압축 푼 폴더에서 **`설치.bat` 를 다시 더블클릭**하면 됩니다.

---

## 5단계. 우분투 이름·비밀번호 만들기 (한 번만)

재부팅 후 **우분투(Ubuntu) 창**이 뜨면서 이렇게 물어봅니다:

- `Enter new UNIX username:` → 영문 소문자로 이름 입력 (예: `student`) → Enter
- `New password:` → 비밀번호 입력 (화면에 안 보이는 게 정상) → Enter
- `Retype new password:` → 같은 비밀번호 다시 → Enter

> ✍️ 이 이름·비밀번호는 나중에 가끔 필요하니 **적어두세요.**
> 📷 [스크린샷 자리: 우분투 username/password 입력]

이 설정이 끝나면 **자동으로 나머지(tmux·Claude Code·Codex·강의 도구)가 설치**됩니다. 기다리면 됩니다.

> 🔑 설치 중간에 **`[sudo] password for ...:`** 라고 비밀번호를 물어보면, **방금 만든 우분투 비밀번호**를 입력하고 Enter 하세요. (화면에 안 보이는 게 정상)

---

## 6단계. 로그인 (Claude)

설치가 끝나면 우분투 터미널에서:

1. `claude` 라고 입력하고 Enter.
2. 브라우저가 열리면 **Claude 계정(Pro Max 5x)으로 로그인** → 권한 허용.
3. 터미널로 돌아오면 끝! 🎉

> Codex 로그인은 **1.2.4 / 1.2.5 강의**에서 안내합니다. (앞에서 이미 깔렸으면 그 부분은 지나가셔도 됩니다.)

---

## 자주 묻는 것

| 증상 | 해결 |
|---|---|
| 재부팅 후 설치가 안 이어져요 | 압축 푼 폴더의 `설치.bat` 다시 더블클릭 |
| `claude` 명령이 없다고 나와요 | 우분투 터미널을 **새로 열고** 다시 `claude` |
| 우분투 비밀번호를 까먹었어요 | 우분투 창에서 다시 만들 수 있어요(강사/조교 문의) |
| 중간에 멈췄어요 | `설치.bat` 다시 실행 — 이미 깔린 건 건너뛰고 이어서 합니다 |
